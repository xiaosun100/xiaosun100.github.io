软件公司常见薪资方式
  ```
  基本工资＋绩效＋奖金
  正常签订合同之后公司会为员工缴纳五险一金
    养老 医疗 工伤 失业 生育
    住房公积金
    五险一金一般公司是按照基本工资进行缴纳
    大部分公司实际做法是按照当地的最低标准进行缴纳

  基本工资,常见的组成包括(岗位工资、全勤奖、公交补助、餐补)
  当你和公司签订劳动合同之后,公司的人事或者行政会给到你一本员工手册
  入职之后
    会有一个培训
    然后会给你指定一个人，以后遇到问题可以问那个人
    注意事项：
      为你的个人办公电脑设置密码
      离开座位超过5分钟的情况下电脑必须进入登录界面(待机状态)
      
  ```
